# Release Checklist

## Before Release

- Code Freeze Complete
- Unit Tests Passed
- Integration Tests Passed
- Performance Tests Passed
- Security Scan Passed
- SonarQube Passed
- Release Notes Prepared

---

## Deployment

- Deployment Started
- Health Checks Verified
- Metrics Verified
- Logs Verified

---

## After Deployment

- Smoke Tests Passed
- Dashboards Verified
- Alerts Verified
- Customer Impact Checked

---

## Final Status

Approved / Rejected
