# ADR Governance Policy

## Mandatory ADR Triggers

ADR required for:

- New Technology
- New Database
- New Messaging System
- Security Architecture Changes
- Multi Region Changes
- Multi Tenant Changes
- ML Platform Decisions

---

## Approval Process

Developer
    ↓
ADR Draft
    ↓
Architecture Review Board
    ↓
Approval
    ↓
Implementation

---

## ADR Status Values

Proposed

Under Review

Approved

Deprecated

Rejected

Superseded

---

## Retention

ADRs must never be deleted.

Deprecated ADRs remain auditable.
