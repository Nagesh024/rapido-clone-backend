# ADR-003

## Title

PostgreSQL Standard Database

## Status

Approved

## Context

Need enterprise-grade relational database.

## Decision

Adopt PostgreSQL as standard database.

Extensions:

- PostGIS
- pg_stat_statements

## Alternatives Considered

MySQL

MariaDB

Oracle

## Consequences

Pros:

- ACID Compliance
- High Availability
- Advanced Querying

Cons:

- Requires DBA Expertise
