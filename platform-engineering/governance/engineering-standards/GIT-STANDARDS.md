# Git Standards

## Branch Naming

feature/ride-booking

feature/payment-retry

bugfix/token-refresh

hotfix/payment-failure

---

## Commit Format

Correct:

feat: add ride booking API

fix: resolve payment retry issue

refactor: improve driver matching logic

Incorrect:

update code

changes

fixed bug

---

## Pull Requests

Minimum 1 Reviewer

Critical Services:

2 Reviewers Required
