# Architecture Governance Policy

## Mandatory Reviews

The following require ARB approval:

- New Microservice
- New Database Technology
- New Messaging Technology
- New Cloud Service
- Security Architecture Changes
- FinTech Architecture Changes
- Multi-Tenant Changes
- Multi-Region Changes

---

## Review Gates

Gate 1:
Architecture Review

Gate 2:
Security Review

Gate 3:
Risk Assessment

Gate 4:
Approval

Gate 5:
Implementation

---

## Non Compliance

Any implementation without approval:

- Cannot be promoted to Production
- Will fail Governance Certification
- Must undergo Retroactive Review
