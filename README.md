\# Rapido Clone Backend



Production-ready Java Spring Boot microservices backend for a Rapido-like ride booking platform.



\## Tech Stack



\- Java 17

\- Spring Boot 3.x

\- PostgreSQL

\- Docker

\- Docker Compose

\- Maven

\- REST APIs

\- Spring Security



\## Current Services



\- auth-service



\## Run Project



```bash

docker compose up --build

```



\## Health API



```bash

http://localhost:8081/health

```
